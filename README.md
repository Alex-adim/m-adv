# 🌟 Math Adventure!

A fun, colorful arithmetic web game for first graders — no installation needed, just open `index.html` in any browser!

## Features

- **Level 1** — Addition & subtraction with numbers 1–10
- **Level 2** — Addition, subtraction & multiplication with numbers 1–20
- **Level 3** — Word problems (addition & subtraction up to 20)
- Multiple-choice answers (4 options per question)
- 5 questions per round with a score at the end
- 🔥 Streak counter, confetti bursts, floating stars, and fun animations
- 🪰 Bonus fly — catch it for extra points!
- 🎬 Video rewards for perfect scores
- 🏆 High score leaderboard (stored locally)
- Celebration screen at the end with a score summary
- Fully responsive — works on tablets and phones too

## How to Play

1. Open `index.html` in any web browser
2. Enter your name
3. Choose **Level 1** (numbers 1–10), **Level 2** (numbers 1–20), or **Level 3** (word problems)
4. Tap or click the correct answer
5. Finish all 5 questions to see your score!

## Tech

Pure HTML, CSS, and vanilla JavaScript — no dependencies, no build step.
